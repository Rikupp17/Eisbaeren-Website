Eine minimalistische Website über Eisbären – erstellt als Übungsprojekt mit **HTML, CSS und JavaScript**.

## 🌍 Inhalte

* Lebensraum der Eisbären
* Alter, Paarungszeit & Ernährung
* Aktuelle Bestandszahlen
* Situation des Lebensraums
* Eisbären in deutschen Zoos

## 💻 Technologien

* HTML5
* CSS3
* JavaScript (für kleine Animationen)

## 🎯 Ziel

Dieses Projekt dient zum Üben von sauberem Webdesign, Layouts und modernen UI-Strukturen.

## 📸 Features

* Modernes, minimalistisches Design
* Responsive Layout
* Smooth Scroll & kleine Animationen
* Bilder & GIFs für bessere Darstellung

## 🚀 Nutzung

Einfach die `index.html` öffnen – keine Installation nötig.
